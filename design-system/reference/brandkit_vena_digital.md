# Brand Kit Vena Digital v1.0

## 1. Resumen estratégico

**Marca:** Vena Digital  
**Idea madre:** IA práctica que funciona para negocios reales.  
**Concepto visual:** Interfaz humana.  
**Personalidad:** cercana, ingeniosa, práctica, clara y enérgica.  
**Promesa visual:** hacer que la inteligencia artificial se sienta útil, entendible, accionable y cercana para emprendedores y marcas personales.

Vena Digital no debe verse como una marca futurista fría, una agencia corporativa aburrida ni una academia técnica complicada. Debe sentirse como una marca moderna, clara y con carácter, capaz de traducir la IA en acciones concretas para negocios reales.

---

## 2. Esencia de marca

### Idea madre

> **IA práctica que funciona para negocios reales.**

Esta frase debe guiar el sistema visual, verbal y de contenido. Todo lo que se diseñe para Vena Digital debe comunicar utilidad real, claridad y aplicación inmediata.

### Concepto visual

> **Interfaz humana**

Una marca clara, práctica y cercana para enseñar y aplicar IA sin fricción.

### Personalidad de marca

| Rasgo | Descripción |
|---|---|
| **Cercana** | Hablamos como personas, no como máquinas. |
| **Ingeniosa** | Transformamos ideas complejas en soluciones simples. |
| **Práctica** | Enseñamos lo que funciona en el mundo real. |
| **Clara** | Comunicación sencilla, estructurada y directa. |
| **Enérgica** | Motivamos a actuar con confianza y entusiasmo. |

### Debe sentirse

- Tecnología fácil.
- Sistema y orden.
- Cercanía humana.
- Claridad visual.
- Energía práctica.
- Marca propia, no genérica.

### Evitar

- Futurismo frío.
- Agencia corporativa aburrida.
- Estética vendehumo.
- Visuales recargados.
- Doodles infantiles o decorativos sin función.
- Exceso de stickers o elementos cute.
- Composiciones caóticas.
- Gradientes invasivos.

## 3. Sistema de logo

### Logo principal

El logo principal integra:

- Mascota/personaje amarillo.
- Gesto de paz.
- Chompa/canguro deportivo coral.
- Círculo o contenedor tipo emblema.
- Wordmark **Vena Digital**.
- Azul noche como color estructural.
- Coral como color de acción.

El logo tiene una personalidad cercana, energética y memorable. Funciona como una mezcla entre marca educativa, identidad digital y sistema de contenido con mascota propia.

### Versiones del logo

#### 1. Horizontal

Uso recomendado para:

- Sitio web.
- Presentaciones.
- Encabezados.
- Firma de piezas.
- Material institucional.

#### 2. Isotipo

Uso recomendado para:

- Avatar.
- Favicon.
- Stickers.
- Íconos de comunidad.
- Miniaturas.
- Sello visual.

#### 3. Sello / avatar

Uso recomendado para:

- WhatsApp.
- Comunidad.
- Recursos descargables.
- Portadas de clases.
- Elementos de marca en carruseles.

### Área de protección

El espacio libre mínimo alrededor del logo debe ser igual o mayor a:

- La altura de la mano del personaje.
- O la altura de la palabra **Vena** en el wordmark.

No se deben ubicar textos, íconos ni otros elementos visuales demasiado cerca del logo. Sí, incluso si “hay espacio”, esa excusa ya ha destruido suficientes marcas.

### Tamaño mínimo

| Medio | Tamaño mínimo recomendado |
|---|---:|
| Impreso | 25 mm de ancho mínimo |
| Digital | 120 px de ancho mínimo |

### Uso correcto

- Mantener alto contraste.
- Usar aire alrededor.
- Respetar proporciones.
- Priorizar fondos claros.
- Mantener buena legibilidad del texto.
- Usar la versión horizontal cuando haya espacio suficiente.
- Usar el isotipo cuando el espacio sea reducido.

### Evitar

- Deformar el logo.
- Cambiar los colores sin criterio.
- Usar degradados invasivos.
- Saturar de detalles alrededor.
- Perder legibilidad.
- Usarlo sobre fondos demasiado cargados.
- Encerrar el logo en formas que compitan con su estructura.

## 4. Paleta de color

La paleta combina energía, cercanía y estructura. El azul da seriedad y orden. El coral activa. El amarillo aporta energía. Los fondos claros mantienen legibilidad y cercanía.

### Paleta principal

| Color | Hex | Rol |
|---|---:|---|
| **Azul noche** | `#0A2142` | Color principal, estructura, títulos, fondos fuertes, botones secundarios. |
| **Coral acción** | `#FF6B5E` | CTAs, énfasis, acentos, palabras clave. |
| **Amarillo energía** | `#F5C935` | Energía, destacados, chips, elementos del personaje. |
| **Marfil digital** | `#F7F5EF` | Fondo cálido principal. |
| **Gris niebla** | `#ECE9E2` | Fondos secundarios, divisores, áreas suaves. |
| **Blanco interfaz** | `#FFFFFF` | Tarjetas, ventanas, contenedores, respiración visual. |

### Paleta de apoyo

| Color | Hex | Rol |
|---|---:|---|
| **Azul suave** | `#BFD7EA` | Fondos secundarios, tarjetas informativas, detalles de apoyo. |
| **Gris carbón** | `#2D2D2D` | Texto alternativo, contraste, detalles neutros. |
| **Verde sistema** | `#5FAE8B` | Confirmaciones, checks, estados positivos. |
| **Coral suave** | `#F4A299` | Fondos suaves, acentos cálidos, detalles complementarios. |

### Jerarquía de uso

| Grupo de color | Uso aproximado |
|---|---:|
| Neutrales: blanco, marfil, gris niebla | 50% |
| Azul noche | 22% |
| Coral acción | 14% |
| Amarillo energía | 8% |
| Colores de apoyo | 6% |

### Regla de uso

> El azul estructura. El coral activa. El amarillo aporta energía. Los fondos claros mantienen cercanía y legibilidad.

### Evitar

- Usar todos los acentos al mismo tiempo.
- Oscurecer demasiado la marca.
- Convertir el color en ruido.
- Usar amarillo como fondo dominante en exceso.
- Usar coral para todo lo importante, porque entonces nada será importante. Diseño, esa ciencia cruel.

## 5. Sistema tipográfico

### Tipografías principales

| Fuente | Uso | Peso recomendado |
|---|---|---|
| **Manrope** | Títulos y jerarquía principal | Bold / ExtraBold |
| **DM Sans** | Cuerpo de texto y explicación | Regular / Medium |
| **IBM Plex Mono** | Etiquetas, tags, acentos UI y notas técnicas | Medium / SemiBold |

### Jerarquía recomendada

| Elemento | Tamaño | Estilo | Uso |
|---|---:|---|---|
| **H1** | 56 px | Bold | Títulos principales. |
| **H2** | 32 px | Bold | Subtítulos. |
| **H3** | 24 px | SemiBold | Secciones. |
| **Body** | 16 px | Regular | Párrafos y texto general. |
| **Caption** | 12 px | Medium | Notas, pies de foto y microcopy. |

### Reglas tipográficas

- Titulares cortos y directos.
- Mucho aire entre bloques.
- IBM Plex Mono solo como acento.
- Priorizar legibilidad sobre adorno.
- Evitar textos largos en portadas.
- Usar contraste fuerte entre título, subtítulo y cuerpo.
- Mantener un ritmo editorial limpio.

### Principio

> **Claro > complicado**

Esta regla debe aplicarse tanto al diseño como al contenido. No estamos haciendo un catálogo de fuegos artificiales tipográficos, por suerte.

## 7. Fotografía y composición

La fotografía de Vena Digital debe conectar con la figura de Laura como guía práctica y cercana, sin convertir cada pieza en una pose corporativa rígida.

### Dirección fotográfica

- Laura enseñando o explicando.
- Momentos casuales y cercanos.
- Herramientas reales: laptop, celular, libreta.
- Luz limpia y fondos simples.
- Ambientes reales de trabajo.
- Expresión natural, no excesivamente posada.
- Mirada directa cuando el contenido necesite confianza.

### Tratamiento visual

- Fotos dentro de ventanas o tarjetas.
- Bordes redondeados.
- Fondo claro.
- Detalles UI mínimos.
- Recortes limpios.
- Integración con bloques de texto.
- Uso de curvas o contenedores para enmarcar.

### Composición base

- Mucho aire.
- Una idea principal por pieza.
- Titular fuerte.
- Un recurso visual útil.
- Contraste claro entre texto e imagen.
- Foto como apoyo, no como ruido.

### Aplicaciones fotográficas

#### Portada educativa

Uso de foto + titular claro.  
Ejemplo:

> IA práctica que funciona para negocios reales.

#### Opinión con foto

Uso de Laura como voz de autoridad.  
Ejemplo:

> La tecnología solo tiene sentido cuando resuelve problemas reales de las personas.

#### Pieza sistema / flujo

Uso de foto junto a pasos o metodología.  
Ejemplo:

1. Entendemos.  
2. Diseñamos.  
3. Implementamos.

### Evitar

- Fotos rígidas o muy posadas.
- Fondos saturados.
- Exceso de efectos.
- Composiciones sin jerarquía.
- Recortes agresivos sin intención.
- Fotos que compitan con el mensaje.

### Principio

> La foto acompaña la claridad: no compite con el mensaje.

### Tono editorial

- Directo.
- Claro.
- Con humor inteligente.
- Sin tecnicismos innecesarios.
- Accionable.
- Cercano.
- Estratégico.
- Sin humo.

### Estructura de una pieza

1. **Gancho**  
   Captamos la atención.

2. **Problema real**  
   Identificamos algo con lo que conectan.

3. **Explicación útil**  
   Aportamos claridad y valor real.

4. **Acción concreta**  
   Indicamos qué pueden hacer ahora.

### Checklist de contenido

- Una idea principal.
- Un titular fuerte.
- Un ejemplo real.
- Un cierre accionable.
- Diseño limpio.
- Lenguaje simple.
- Aplicación concreta.
- CTA claro cuando corresponda.

### Ejemplo de portada

> Cómo usar IA sin complicarte

Subtexto:

> IA práctica que funciona para negocios reales.

### Principio editorial

> Explicar bien es parte de la marca.

## 13. Reglas de composición

### Estructura recomendada para piezas

1. Etiqueta pequeña.
2. Titular grande.
3. Recurso visual útil: foto, mascota, UI card, flujo o checklist.
4. Acento de color.
5. CTA o cierre.

### Reglas generales

- Una idea principal por pieza.
- Máximo dos colores protagonistas por diseño.
- Mantener márgenes amplios.
- Evitar saturar con íconos.
- Usar coral solo para activar o enfatizar.
- Usar amarillo solo para energía o destaque puntual.
- Usar azul noche para estructura y autoridad.
- Usar fondos claros como base principal.

## 16. Checklist rápido antes de publicar una pieza

Antes de publicar cualquier diseño de Vena Digital, revisar:

- ¿Se entiende en menos de 3 segundos?
- ¿Tiene una sola idea principal?
- ¿El titular es claro y fuerte?
- ¿El diseño tiene suficiente aire?
- ¿El color ayuda o está decorando por ansiedad?
- ¿La foto o la mascota aportan al mensaje?
- ¿El CTA es concreto?
- ¿Suena como Vena Digital y no como una plantilla reciclada?
- ¿Tiene utilidad real para un emprendedor?
- ¿Evita el humo, el futurismo vacío y el exceso visual?
