# Slides — Vena Digital

A 6-slide deck template demonstrating the brand applied at presentation scale (1920×1080). Uses `deck-stage.js` for scaling, keyboard nav, and print-to-PDF.

## Slides
1. **Title** — marfil bg, big headline, mascot card with rotated yellow tag
2. **Problem** — azul noche, large H1 with coral accent word
3. **Pillars** — marfil, four numbered cards
4. **Big quote** — coral fill, displayed quote with yellow accent word
5. **Method** — marfil with three colored step cards (white / azul / amarillo)
6. **CTA** — azul noche, large headline + features panel

## Layout system used
- Eyebrow (mono + dot) sets the section
- H1 is 80–124px Manrope ExtraBold, with one accent word in coral
- Background rotation: marfil → azul → marfil → coral → marfil → azul (avoids monotony)
- Footer line is mono uppercase 18px, low-opacity

## Run
Open `index.html` and use ←/→ to navigate.
