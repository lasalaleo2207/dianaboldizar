# Academia — Vena Digital

Course/lesson UI for "Academia IA para Emprendedores."

## Components
- `Sidebar` — left rail with nav and a downloadable resource pin
- `Topbar` — search + notifications + avatar
- `ProgressCard` — dark blue header card with current module and progress bar
- `ClassCard` — module thumbnail card; supports done/locked/active states
- `ModulesGrid` — 3-column grid of class cards with tab filters
