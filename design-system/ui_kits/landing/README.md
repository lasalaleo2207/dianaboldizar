# Landing — Vena Digital

Marketing site for Vena Digital. The home page demonstrates the full system: nav, hero, pillars, academia teaser, testimonial, CTA, footer.

## Components
- `Nav` — sticky top nav, lockup left, links center, primary + tertiary actions right
- `Hero` — headline + Laura photo card with rotated yellow chip
- `Pillars` — 4-card "Menos caos, más sistema" grid
- `AcademiaTeaser` — dark blue section with module cards
- `Testimonial` — large pull-quote card
- `CTA` — coral closing block
- `Footer` — dark blue, links, copyright

## Run
Open `index.html`. Components depend on `shared/Brand.jsx` (mascot, wordmark, icons) and `shared/components.css`.

## Substitutions flagged
- Laura's photo is a circular placeholder (`FOTO LAURA`).
- Mascot is a placeholder SVG — swap when real artwork ships.
